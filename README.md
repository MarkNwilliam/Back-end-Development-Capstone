# Back-end-Development-Capstone

Django main application for the Back-End Development Capstone project.

## Apps

- **Songs** - Song collection with lyrics modal
- **Photos** - Photo gallery
- **Concerts** - Concert schedule

## Setup

```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
